El presente proyecto trata sobre una aplicación web que permita controlar las ventas físicas y virtuales de una empresa de manufactura para poder realizar el correspondiente seguimiento. Por otro lado, el stack tecnológico que se va a utilizar está relacionado con HTML5, CSS3, JavaScript, React JS, Node JS, OAuth2, entre otras.

Además, se van a contar con los siguientes módulos:

- Módulo de administrador de ventas.
- Módulo de registro del estado de la venta.
- Módulo de gestión de vendedores.
- Módulo de gestión de ingreso al sistema de información.
- Módulo de gestión de usuarios y roles.
